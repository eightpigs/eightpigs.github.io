终端模拟器

Windows
  • Cmder
  • WSL
  • Hyper(Electron)
MacOS
  • Iterm2 + zsh(oh-my-zsh)
  • Hyper(Electron)

查找

Windows
  • Everything
MacOS
  • Alfred
  • Shell Command

快速启动

Windows
  • Listary (可以配合EveryThing）
  • Wox（插件多）
MacOS
  • Alfred

快捷键（自动化）

Windows
  • AHK （AutoHotKey）
MacOS
  • Hammerspoon (Lua脚本)
  • Keyboard Maestro（快捷键 + 自动化）
